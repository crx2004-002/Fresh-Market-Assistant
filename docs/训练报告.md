# 水果识别模型训练报告

## 一、项目概述

### 1.1 项目名称
Fresh Market Assistant（水果智能识别系统）

### 1.2 项目目标
基于深度学习技术，实现39类水果的高精度图像分类，支持实时摄像头识别。

### 1.3 技术栈
- **深度学习框架**: PyTorch 2.0+
- **模型架构**: ConvNeXtV2-Tiny
- **预训练权重**: ImageNet-22K
- **训练环境**: conda yolov (Python 3.12.13)
- **硬件**: RTX 4070 Laptop GPU 8GB

---

## 二、数据集

### 2.1 数据集来源
- **原始数据集**: 10类水果，3,510张图片
- **Kaggle数据集**: 39类水果，39,414张图片
- **合并数据集**: 39类水果，39,986张图片

### 2.2 数据集划分
| 数据集 | 图片数量 | 类别数 | 用途 |
|--------|----------|--------|------|
| 训练集 | 31,648 | 39 | 模型训练 |
| 验证集 | 3,871 | 39 | 训练过程验证 |
| 测试集 | 3,852 | 39 | 最终评估 |

### 2.3 水果类别 (39类)
```
apple, apricot, avocado, banana, bayberry, blueberry, cantaloupe, carambola,
cherry, coconut, cranberry, dragonfruit, durian, fig, grape, grapefruit,
guava, hawthorn, jackfruit, kiwi fruit, kumquat, lemon, longan, loquat,
lychee, mandarine, mango, mulberry, orange, peach, pear, persimmon,
pineapple, plumcot, pomegranate, pomelo, strawberry, tomato, watermelon
```

### 2.4 数据预处理
- **图片尺寸**: 288×288
- **数据增强**: light模式
  - RandomResizedCrop(288)
  - RandomHorizontalFlip
  - ColorJitter(brightness=0.2, contrast=0.2)
- **类别权重**: 启用 (解决数据不平衡)

---

## 三、模型架构

### 3.1 模型选择
**ConvNeXtV2-Tiny** (2023年微软SOTA CNN)

### 3.2 模型参数
| 参数 | 值 |
|------|-----|
| 总参数量 | 28,280,231 |
| 可训练参数 | 28,280,231 (微调阶段) |
| 输入尺寸 | 288×288×3 |
| 输出类别 | 39 |
| 预训练权重 | ImageNet-22K |

### 3.3 模型特点
- 大卷积核 (7×7): 更大视野，捕获更多上下文
- 深度可分离卷积: 更高效地提取特征
- 注意力机制: 自动关注重要区域
- LayerNorm: 稳定训练

## 四、训练配置

### 4.1 训练策略
采用**两阶段训练**策略：
1. **冻结训练**: 冻结骨干网络，只训练分类头
2. **微调**: 解冻骨干网络，精细调整所有参数

### 4.2 冻结训练配置
| 配置项 | 值 |
|--------|-----|
| 冻结骨干 | 是 |
| 学习率 | 0.001 |
| 批次大小 | 32 |
| 训练轮数 | 50 (早停) |
| 优化器 | AdamW |
| 学习率调度 | Cosine Annealing |
| 标签平滑 | 0.1 |
| 权重衰减 | 0.01 |
| AMP混合精度 | 启用 |

### 4.3 微调配置
| 配置项 | 值 |
|--------|-----|
| 冻结骨干 | 否 (全部解冻) |
| 学习率 | 0.0001 |
| 批次大小 | 16 |
| 训练轮数 | 30 |
| 优化器 | AdamW |
| 学习率调度 | Cosine Annealing |
| 标签平滑 | 0.1 |
| 权重衰减 | 0.01 |
| AMP混合精度 | 启用 |
| 基础模型 | 冻结训练最佳模型 (94.82%) |

## 十、关键发现

### 10.1 训练过程中的"相变"现象
在第19轮冻结训练中，验证准确率从92.18%突然跃升到94.26%，这是深度学习中常见的"相变"现象，说明模型突然学会了区分相似水果的关键特征。

### 10.2 微调效果显著
微调将验证准确率从94.82%提升到97.03%，提升了2.21%，证明解冻骨干网络并使用更小学习率进行精细调整是有效的。

### 10.3 模型泛化能力强
测试准确率(97.14%)略高于验证准确率(97.03%)，说明模型泛化能力优秀，没有过拟合。
