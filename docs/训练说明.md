# 水果识别模型 - 训练说明

## 📋 项目概述

本项目使用深度学习进行15类水果的图像识别，支持首次训练和二次微调。

## 🍎 首次训练

### 运行命令
```bash
python train.py
```

### 训练配置
- **模型架构**: ResNet50
- **训练轮数**: 30轮
- **学习率**: 0.001
- **批次大小**: 32
- **冻结策略**: 只训练分类层 (FREEZE_BACKBONE=True)

### 预期结果
- 训练时间: ~7分钟 (RTX 4070)
- 验证集准确率: ~95%

---

## 🔧 二次微调训练

### 运行命令
```bash
python train_finetune.py
```

### 微调配置
- **微调模型**: ./checkpoints/best_model.pth (首次训练的最佳模型)
- **训练轮数**: 20轮
- **学习率**: 0.0001 (比首次小10倍)
- **解冻策略**: 渐进式解冻
  - 解冻最后1个block (layer4) + 分类层 (fc)
  - 更多层可以学习更精细的特征

### 渐进式解冻说明
```
ResNet50 结构:
┌─────────────────────────────────────┐
│  conv1 + bn1 + layer1 + layer2     │  ← 冻结 (学习通用特征)
│  layer3 (中间层)                    │  ← 冻结
│  layer4 (高层特征)                  │  ← 解冻 (学习特定特征) ✅
│  fc (分类层)                        │  ← 解冻 ✅
└─────────────────────────────────────┘
```

### 调整解冻级别
在 `config.py` 中修改 `FINETUNE_UNFREEZE_LEVEL`:
- `0`: 只解冻分类层 (最保守)
- `1`: 解冻最后1个block + 分类层 (推荐)
- `2`: 解冻最后2个block + 分类层
- `-1`: 全部解冻 (最激进)

## 🎯 测试脚本

### 1. 摄像头实时识别
```bash
python camera_realtime.py
```
- 按 `q` 退出
- 按 `s` 截图保存
- 按 `r` 重置稳定器

### 2. 单张图片识别
```bash
# 直接指定图片
python predict_single.py test.jpg

# 交互模式
python predict_single.py
```

### 3. 批量图片识别
```bash
python predict.py --dir ./predict_images --output results.json
```

## ⚠️ 注意事项

1. **显存要求**: RTX 4070 8GB 或更高
2. **首次训练**: 必须先完成首次训练，才能进行二次微调
3. **备份模型**: 二次微调会覆盖 `best_model.pth`，建议先备份
4. **训练时间**: 二次微调约需5分钟
