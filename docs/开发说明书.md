# 🍎 水果智能识别 App 开发说明书

## 一、项目概述

### 1.1 项目名称
水果智能识别 App (FruitLens)

### 1.2 项目目标
开发一款基于 Android 原生的水果识别应用，集成端侧 AI 模型进行实时水果识别，并接入 DeepSeek API 实现水果科普和智能聊天功能。

### 1.3 核心功能
| 功能 | 说明 |
|------|------|
| 📸 实时识别 | 调用摄像头实时预览，端侧模型识别15类水果 |
| 🖼️ 批量识别 | 支持从相册选择多张图片批量识别 |
| 💬 AI 聊天 | 接入 DeepSeek API，实现水果知识问答和智能对话 |
| 📋 历史记录 | 完整保存识别图片、结果、时间，支持查看和删除 |
| 📴 离线优先 | 端侧模型支持离线识别，无网络也可使用核心功能 |

### 1.4 项目范围
- **第一阶段**：Android 原生应用开发（2周）
- **第二阶段**：iOS 应用开发（后续规划）

### 5.1 模型转换流程

#### TFLite 服务实现
```kotlin
class TFLiteService @Inject constructor(
    @ApplicationContext private val context: Context
) {
    private var interpreter: Interpreter? = null
    private val fruitLabels = listOf(
        "apple", "banana", "grape", "orange", "strawberry",
        // ... 其他水果标签
    )

    fun loadModel() {
        val modelFile = loadModelFile("fruit_model.tflite")
        val options = Interpreter.Options().apply {
            setNumThreads(4)
            addDelegate(GpuDelegate()) // 可选：GPU 加速
        }
        interpreter = Interpreter(modelFile, options)
    }

    fun classify(bitmap: Bitmap): List<FruitResult> {
        // 预处理：resize 到 224x224，归一化
        val input = preprocessImage(bitmap)
        val output = Array(1) { FloatArray(15) }

        // 推理
        interpreter?.run(input, output)

        // 后处理：softmax + 过滤
        return postprocess(output[0])
    }

    private fun preprocessImage(bitmap: Bitmap): ByteBuffer {
        val resized = Bitmap.createScaledBitmap(bitmap, 224, 224, true)
        val buffer = ByteBuffer.allocateDirect(4 * 224 * 224 * 3)
        buffer.order(ByteOrder.nativeOrder())

        val pixels = IntArray(224 * 224)
        resized.getPixels(pixels, 0, 224, 0, 0, 224, 224)

        for (pixel in pixels) {
            buffer.putFloat(((pixel shr 16) and 0xFF) / 255.0f) // R
            buffer.putFloat(((pixel shr 8) and 0xFF) / 255.0f)  // G
            buffer.putFloat((pixel and 0xFF) / 255.0f)           // B
        }

        return buffer
    }

    private fun postprocess(output: FloatArray): List<FruitResult> {
        // Softmax
        val expOutput = output.map { exp(it.toDouble()) }
        val sumExp = expOutput.sum()
        val probabilities = expOutput.map { (it / sumExp).toFloat() }

        // 过滤
        return probabilities.mapIndexed { index, confidence ->
            FruitResult(
                name = fruitLabels[index],
                nameCn = FruitNames.getNameCn(fruitLabels[index]),
                confidence = confidence
            )
        }
        .filter { it.confidence >= 0.70f }
        .sortedByDescending { it.confidence }
        .take(3)
    }

    fun close() {
        interpreter?.close()
    }
}
```

### 6.3 流式输出实现
```kotlin
class ChatRepository @Inject constructor(
    private val api: DeepSeekApi
) {
    suspend fun streamChat(messages: List<Message>): Flow<String> = flow {
        val request = ChatRequest(
            model = DeepSeekConfig.MODEL,
            messages = messages,
            stream = true
        )

        val response = api.chatStream(request)
        val reader = response.byteStream().bufferedReader()

        try {
            while (true) {
                val line = reader.readLine() ?: break
                if (line.startsWith("data: ")) {
                    val data = line.removePrefix("data: ")
                    if (data == "[DONE]") break

                    val chunk = Gson().fromJson(data, ChatChunk::class.java)
                    val content = chunk.choices.firstOrNull()?.delta?.content
                    if (content != null) {
                        emit(content)
                    }
                }
            }
        } finally {
            reader.close()
            response.close()
        }
    }
}
```

## 九、注意事项

### 9.1 性能优化
- **模型推理**：使用 GPU Delegate 加速（可选）
- **图片处理**：在后台线程处理，避免阻塞 UI
- **内存管理**：及时释放 Bitmap 和模型资源
- **电池优化**：非活跃时暂停相机预览

### 9.2 权限处理
```xml
<!-- AndroidManifest.xml -->
<uses-permission android:name="android.permission.CAMERA" />
<uses-permission android:name="android.permission.INTERNET" />
<uses-permission android:name="android.permission.READ_MEDIA_IMAGES" /> <!-- Android 13+ -->
<uses-permission android:name="android.permission.READ_EXTERNAL_STORAGE" /> <!-- Android 12- -->
```

### 9.3 API Key 安全
- 使用 EncryptedSharedPreferences 存储
- 提供设置页面让用户输入 API Key
- 不要硬编码在代码中

### 9.4 离线能力
- 端侧模型支持离线识别
- 历史记录本地存储
- AI 聊天需要网络

### 9.5 兼容性
- **最低 SDK**：Android 8.0 (API 26)
- **目标 SDK**：Android 14 (API 34)
- **测试设备**：建议中端及以上手机

## 附录：快速开始

### 1. 环境要求
- Android Studio Hedgehog (2023.1.1) 或更高版本
- JDK 17
- Android SDK 34

### 2. 配置 API Key
1. 申请 DeepSeek API Key：https://platform.deepseek.com/
2. 在 App 设置页面输入 API Key

### 3. 运行项目
```bash
# 克隆项目
git clone <repository-url>

# 打开 Android Studio
# File -> Open -> 选择项目目录

# 运行
# 点击 Run 按钮或 Shift + F10
```

### 4. 构建 APK
```bash
# Debug 版本
./gradlew assembleDebug

# Release 版本
./gradlew assembleRelease
```
