# Flutter 安装指南（Windows）

## 第一步：安装 Android Studio

1. 下载 Android Studio：
   https://developer.android.com/studio

2. 安装时勾选：
   - Android SDK
   - Android SDK Platform-Tools
   - Android Virtual Device (AVD)

3. 安装完成后打开 Android Studio，完成初始化向导

4. 安装 Android SDK：
   - 打开 Android Studio → Settings → Languages & Frameworks → Android SDK
   - 安装 Android 14 (API 34)
   - 安装 Android SDK Build-Tools
   - 安装 Android SDK Platform-Tools

## 第二步：下载 Flutter SDK

1. 下载 Flutter SDK：
   https://docs.flutter.dev/get-started/install/windows

2. 解压到：`C:\flutter`

3. 添加环境变量：
   - 用户 PATH 中添加：`C:\flutter\bin`

## 第三步：配置 Flutter

```bash
# 打开新的命令行窗口，运行：
flutter doctor

# 按照提示修复问题
flutter doctor --android-licenses  # 接受 Android 许可证
```

## 第四步：创建项目

```bash
cd E:\progect\实训
flutter create fruit_lens
cd fruit_lens
```

## 常见问题

### Q: flutter doctor 报错 "Android toolchain - develop for Android devices"
A: 确保 Android SDK 路径正确，运行：
```bash
flutter config --android-sdk C:\Users\<你的用户名>\AppData\Local\Android\Sdk
```

### Q: 模拟器无法启动
A: 在 Android Studio → Tools → Device Manager → 创建虚拟设备
